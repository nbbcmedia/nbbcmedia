# I/O for the Mackie DL1608
{: .new-title }
> NBBC Specific Information
>
> We typically only focus on these 6 inputs:
> 1. Piano Bass
> 2. Piano Treble
> 3. Piano Vox
> 4. Pulpit
> 5. Lavalier
> 6. Computer
## Inputs
The Mackie DL1608 has a total of 16 XLR inputs, 4 of which are XLR/TRS combined inputs.
